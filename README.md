## 🎵 Spotify Clone

A fully responsive and user-friendly Spotify Clone built using React.js, JavaScript, Tailwind CSS, CSS, and HTML.

# 🚀 Features : -

1. 🎧 Clean and modern UI inspired by Spotify

2. 📱 Fully responsive design – works seamlessly on all screen sizes

3. ⚡ Built with React.js for fast and interactive performance

4. 🌈 Styled using Tailwind CSS for a consistent and elegant layout

5. 💡 Easy to navigate and intuitive user experience

# 🛠️ Tech Stack

1. React.js

2. JavaScript

3. Tailwind CSS

4. HTML & CSS
